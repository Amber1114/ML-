# 财务健康评估指标体系

## 1. 收支结构评估（30分）

### 1.1 储蓄率（15分）
- 优秀：储蓄率 ≥ 30% → 15分
- 良好：20% ≤ 储蓄率 < 30% → 12分
- 一般：10% ≤ 储蓄率 < 20% → 8分
- 较差：5% ≤ 储蓄率 < 10% → 4分
- 很差：储蓄率 < 5% → 0分

### 1.2 收入稳定性（10分）
- 优秀：过去6个月收入波动 < 10% → 10分
- 良好：过去6个月收入波动 10%-20% → 8分
- 一般：过去6个月收入波动 20%-30% → 5分
- 较差：过去6个月收入波动 30%-50% → 3分
- 很差：过去6个月收入波动 > 50% → 0分

### 1.3 消费占比合理性（5分）
- 优秀：消费占收入比例 < 50% → 5分
- 良好：50% ≤ 消费占收入比例 < 70% → 4分
- 一般：70% ≤ 消费占收入比例 < 90% → 2分
- 较差：90% ≤ 消费占收入比例 < 100% → 1分
- 很差：消费占收入比例 > 100% → 0分

## 2. 资产负债状况（25分）

### 2.1 净资产状况（10分）
- 优秀：净资产 > 3年总收入 → 10分
- 良好：1年总收入 < 净资产 ≤ 3年总收入 → 8分
- 一般：6个月总收入 < 净资产 ≤ 1年总收入 → 5分
- 较差：净资产 > 0 且 ≤ 6个月总收入 → 3分
- 很差：净资产 ≤ 0 → 0分

### 2.2 债务收入比（10分）
- 优秀：债务收入比 < 20% → 10分
- 良好：20% ≤ 债务收入比 < 30% → 8分
- 一般：30% ≤ 债务收入比 < 40% → 5分
- 较差：40% ≤ 债务收入比 < 50% → 3分
- 很差：债务收入比 ≥ 50% → 0分

### 2.3 流动性比率（5分）
- 优秀：流动性比率 ≥ 6 → 5分
- 良好：4 ≤ 流动性比率 < 6 → 4分
- 一般：3 ≤ 流动性比率 < 4 → 3分
- 较差：2 ≤ 流动性比率 < 3 → 2分
- 很差：流动性比率 < 2 → 0分

## 3. 财务安全保障（20分）

### 3.1 应急基金充足度（10分）
- 优秀：应急基金 ≥ 6个月支出 → 10分
- 良好：4个月支出 ≤ 应急基金 < 6个月支出 → 8分
- 一般：3个月支出 ≤ 应急基金 < 4个月支出 → 6分
- 较差：2个月支出 ≤ 应急基金 < 3个月支出 → 3分
- 很差：应急基金 < 2个月支出 → 0分

### 3.2 保险保障程度（10分）
- 优秀：拥有全面保险（重疾+意外+医疗+寿险）→ 10分
- 良好：拥有基本保险（重疾+意外）→ 8分
- 一般：拥有部分保险（意外或医疗）→ 5分
- 较差：只有基本社保 → 2分
- 很差：无任何保险 → 0分

## 4. 消费行为分析（15分）

### 4.1 消费多样性（7分）
- 优秀：消费类别 ≥ 8个 → 7分
- 良好：6 ≤ 消费类别 < 8个 → 6分
- 一般：4 ≤ 消费类别 < 6个 → 4分
- 较差：2 ≤ 消费类别 < 4个 → 2分
- 很差：消费类别 < 2个 → 0分

### 4.2 冲动消费控制（8分）
- 优秀：非计划消费占比 < 10% → 8分
- 良好：10% ≤ 非计划消费占比 < 20% → 6分
- 一般：20% ≤ 非计划消费占比 < 30% → 4分
- 较差：30% ≤ 非计划消费占比 < 40% → 2分
- 很差：非计划消费占比 ≥ 40% → 0分

## 5. 长期财务规划（10分）

### 5.1 投资比例（5分）
- 优秀：投资资产占比 ≥ 30% → 5分
- 良好：20% ≤ 投资资产占比 < 30% → 4分
- 一般：10% ≤ 投资资产占比 < 20% → 3分
- 较差：5% ≤ 投资资产占比 < 10% → 2分
- 很差：投资资产占比 < 5% → 0分

### 5.2 退休储备（5分）
- 优秀：有明确退休计划且定期储备 → 5分
- 良好：有退休计划但偶尔储备 → 3分
- 一般：考虑过退休但无具体计划 → 2分
- 较差：很少考虑退休问题 → 1分
- 很差：从未考虑退休问题 → 0分

## 总评分标准
- 优秀：90-100分
- 良好：80-89分
- 一般：70-79分
- 需改进：60-69分
- 较差：< 60分
